# Технические решения

## Выбор технологий

### Next.js 14+ (App Router)
- **Причина:** Современный фреймворк React с серверными компонентами, встроенной оптимизацией и удобной маршрутизацией
- **App Router:** Используется для лучшей производительности и поддержки React Server Components

### TypeScript
- **Причина:** Строгая типизация обеспечивает надежность кода и предотвращает ошибки на этапе разработки
- Все интерфейсы и типы должны быть явно определены

### Tailwind CSS
- **Причина:** Утилитарный подход к стилизации, быстрая разработка UI, легкая кастомизация
- Используется для верстки всех компонентов

### Howler.js
- **Причина:** Надежная библиотека для работы с Web Audio API
- Поддержка управления громкостью, слоями звука, точной синхронизации
- Кроссбраузерная совместимость

### Zustand
- **Причина:** Легковесный state management без излишней сложности
- Идеально подходит для управления состоянием плеера, настроек и списка треков
- Минимальный boilerplate код

### Prisma + MySQL
- **Причина:** Prisma обеспечивает типобезопасность при работе с БД
- Легкие миграции и удобный API для запросов
- MySQL выбран для совместимости с Timeweb VPS

### Docker
- **Причина:** Контейнеризация упрощает деплой и обеспечивает консистентность окружения
- Легкое развертывание на VPS
